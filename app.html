What is AI?

Introduction to core AI concepts and a small code demonstration.

Concept

Artificial Intelligence (AI) refers to systems that perform tasks which normally require human intelligence. In modern practice this often means statistical models trained on data (e.g., neural networks) that can make predictions or generate content.

"AI is not magic — it's applied statistics, engineering, and careful product design."
  • • Models learn patterns from data
  • • Quality of data strongly influences results
  • • Responsible deployment requires monitoring and guardrails

Key API Parameters

Temperature
0.0 - 1.0

Controls randomness. Lower values make outputs more deterministic; higher values increase creativity and unpredictability.

<\/div>
Example: 0.2 — precise; 0.8 — creative
Top_P (nucleus sampling)
0.0 - 1.0

Limits tokens to a probability mass. Use instead of/in combination with Temperature for more controlled randomness.

<\/div>
Example: 0.9 — broad sampling; 0.1 — narrow sampling

// Simple example: call LLM API (pseudocode)
async function summarize(text) {
  const resp = await fetch('/api/llm', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      prompt: `Summarize the text in one paragraph:\n\n${text}`,
      temperature: 0.2,
      top_p: 0.9
    })
  });
  const data = await resp.json();
  return data.output;
}

// usage
summarize("Long article...").then(console.log);
