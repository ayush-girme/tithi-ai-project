Key API Parameters

Temperature

Controls randomness. Lower values make outputs more deterministic.

Top_P (nucleus sampling)

Limits tokens to a probability mass. Use for controlled randomness.

api.js
async function summarize(text) {
  const resp = await fetch('/api/llm', {
    method: 'POST',
    body: JSON.stringify({
      prompt: `Summarize:\n\n${text}`,
      temperature: 0.2
    })
  });
  return (await resp.json()).output;
}
